
## AI-Powered Resume Screening & Feedback System

TalentRank is an AI-based resume screening system that helps recruiters analyze resumes, match candidates with job requirements, and generate explainable feedback using NLP and Generative AI techniques.

The goal of this project is to reduce manual resume screening time by automatically extracting skills, comparing candidate profiles with job descriptions, and providing meaningful insights for hiring decisions.


## Features

- Resume parsing from PDF, DOCX, and TXT files
- Automatic skill and experience extraction
- Job description and resume matching
- Semantic similarity-based candidate ranking
- Explainable scoring with:
  - Matched skills
  - Missing skills
  - Candidate strengths
  - Improvement areas
- AI-generated resume feedback
- Candidate comparison workflow
- Recruiter-friendly dashboard

## How It Works

1. Recruiter uploads a job description.
2. Candidate resumes are uploaded.
3. The system extracts resume information.
4. NLP models analyze skills, experience, and qualifications.
5. Semantic matching compares resumes with job requirements.
6. The system generates candidate rankings and feedback.
